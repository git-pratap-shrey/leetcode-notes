The n8n workflows are stored as JSON files in the project directory:
- `leetcode-notes-main.json` — main workflow
- `leetcode-notes-authorizer.json` — authorizer workflow

**Why:** User has these workflows available for reference when working on the pipeline.
**How to apply:** When asked about the n8n pipeline or LeetCode automation, read these JSON files for the actual workflow configuration.
